HTML: велика шпаргалка

Довідник по основних HTML-тегах, до якого можна повертатися протягом усього курсу. Тут є каркас сторінки, текстові елементи, списки, зображення, таблиці, форми та корисні правила.

Каркас сторінки

<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Моя сторінка</title>
</head>
<body>
  Тут вміст сторінки
</body>
</html>
Тег Що робить Нотатка
<!DOCTYPE html> Каже браузеру, що це HTML5-документ. Пишеться на самому початку файла.
<html> Огортає всю сторінку. Може мати атрибут lang="uk".
<head> Містить службову інформацію. Тут не показується основний вміст сторінки.
<body> Містить усе, що бачить користувач. Заголовки, текст, картинки, кнопки, списки.

Теги в head

Тег Для чого Приклад
<title> Назва вкладки в браузері. <title>Мій сайт</title>
<meta charset="UTF-8"> Правильне відображення українських літер. <meta charset="UTF-8">
<meta name="viewport"> Щоб сайт нормально виглядав на телефоні. <meta name="viewport" content="width=device-width, initial-scale=1.0">
<link> Підключення CSS-файла. <link rel="stylesheet" href="styles.css">

Текст і заголовки

Тег Для чого Приклад
<h1> - <h6> Заголовки різного рівня. <h1>Головна тема</h1>
<p> Абзац звичайного тексту. <p>Це абзац.</p>
<br> Перенос рядка. Привіт<br>Світ
<hr> Горизонтальна лінія, розділювач. <hr>
<strong> Важливий текст, зазвичай жирний. <strong>Увага</strong>
<em> Наголос, зазвичай курсив. <em>Дуже цікаво</em>
<span> Маленький фрагмент тексту всередині рядка. <span>слово</span>

Посилання і медіа

Тег Для чого Приклад
<a> Посилання на іншу сторінку або сайт. <a href="about.html">Про мене</a>
<img> Зображення. <img src="cat.jpg" alt="Кіт">
<figure> Блок для картинки або схеми. <figure>...</figure>
<figcaption> Підпис до картинки всередині figure. <figcaption>Мій кіт</figcaption> 
<a href="contacts.html">Контакти</a>

<figure>
  <img src="robot.png" alt="Робот">
  <figcaption>Малюнок робота</figcaption>
</figure>

Списки

Тег Для чого Приклад
<ul> Ненумерований список. <ul>...</ul>
<ol> Нумерований список. <ol>...</ol>
<li> Окремий пункт списку. <li>Піца</li> 
<h3>Мої хобі</h3>
<ul>
  <li>Малювання</li>
  <li>Ігри</li>
  <li>Програмування</li>
</ul>

<h3>Кроки</h3>
<ol>
  <li>Відкрити файл</li>
  <li>Написати код</li>
  <li>Зберегти</li>
</ol>

Блоки і структура сторінки

Тег Для чого Коли використовувати
<div> Універсальний блок-контейнер. Коли треба просто об'єднати елементи в один блок.
<header> Верхня частина сторінки або секції. Назва сайту, меню, вступ.
<nav> Блок навігації. Посилання на інші сторінки.
<main> Головний вміст сторінки. Основна частина документа.
<section> Окремий розділ сторінки. Коли є тема і заголовок.
<article> Самостійний матеріал або картка. Новина, пост, стаття, окремий блок.
<footer> Нижня частина сторінки. Автор, контакти, копірайт.

Таблиці

Тег Для чого Приклад
<table> Уся таблиця. <table>...</table>
<tr> Рядок таблиці. <tr>...</tr>
<th> Заголовок стовпця. <th>Ім'я</th>
<td> Звичайна клітинка. <td>Оля</td>
<thead>, <tbody> Частини таблиці. Допомагають краще організувати код. 
<table>
  <thead>
    <tr>
      <th>Ім'я</th>
      <th>Улюблений колір</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Марко</td>
      <td>Синій</td>
    </tr>
  </tbody>
</table>

Форми

Тег Для чого Приклад
<form> Огортає всю форму. <form>...</form>
<label> Підпис до поля. <label for="name">Ім'я</label>
<input> Поле введення. <input type="text" id="name">
<textarea> Поле для довшого тексту. <textarea></textarea>
<button> Кнопка. <button>Надіслати</button> 
<form>
  <label for="name">Твоє ім'я</label>
  <input type="text" id="name" name="name">

  <label for="message">Повідомлення</label>
  <textarea id="message" name="message"></textarea>

  <button type="submit">Надіслати</button>
</form>

Корисні атрибути

Атрибут Для чого Приклад
href Адреса посилання. <a href="about.html">Про мене</a>
src Шлях до картинки або файла. <img src="cat.png" alt="Кіт">
alt Текстовий опис картинки. <img src="cat.png" alt="Кіт у капелюсі">
class Ім'я класу для CSS. <div class="card"></div>
id Унікальне ім'я елемента. <h1 id="title">Привіт</h1>
type Тип поля або кнопки. <input type="email">
name Ім'я поля у формі. <input name="user_name">
placeholder Підказка всередині поля. <input placeholder="Введи ім'я">

Правила HTML, які корисно пам'ятати

  • Більшість тегів мають відкривальний і закривальний запис: <p>...</p>.
  • Деякі теги не мають закривальної частини: <img>, <br>, <hr>, <meta>, <link>.
  • Вкладеність має бути правильною: якщо тег відкрився всередині іншого, то і закритися він має всередині нього.
  • У сторінки зазвичай один головний заголовок <h1>.
  • Для картинки бажано завжди писати alt.
  • HTML відповідає за зміст і структуру, а CSS за зовнішній вигляд.

Повний приклад простої сторінки

<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Мій сайт</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <h1>Клуб юних програмістів</h1>
    <nav>
      <a href="index.html">Головна</a>
      <a href="about.html">Про нас</a>
      <a href="contacts.html">Контакти</a>
    </nav>
  </header>

  <main>
    <section>
      <h2>Чим ми займаємося</h2>
      <p>Ми вивчаємо HTML, CSS і Python Flask.</p>
    </section>

    <section>
      <h2>Наші теми</h2>
      <ul>
        <li>Створення сторінок</li>
        <li>Оформлення сайту</li>
        <li>Робота з веб-сервером</li>
      </ul>
    </section>
  </main>

  <footer>
    <p>2026 Code Academy</p>
  </footer>
</body>
</html>
