Крок 1. Створити файл стилів

Наприклад, файл styles.css.

body {
  background-color: lightyellow;
}

h1 {
  color: darkblue;
}

Крок 2. Підключити файл у HTML

Тег підключення пишемо всередині head.

<head>
  <meta charset="UTF-8">
  <title>Моя сторінка</title>
  <link rel="stylesheet" href="styles.css">
</head>

Типова помилка

Якщо стиль не працює, треба перевірити:

  • чи правильно написано ім'я файлу
  • чи файл лежить у тій самій папці
  • чи є тег link саме в head

Коротко про CSS

“HTML відповідає за те, що є на сторінці. CSS відповідає за те, як це виглядає.”
