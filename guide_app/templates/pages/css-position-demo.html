Відкрити довідник по position

Relative + Absolute

Бейдж усередині картки

absolute

Картка з бейджем

Картка має position: relative, а бейдж усередині неї має position: absolute.

.card {
  position: relative;
}

.badge {
  position: absolute;
  top: 12px;
  right: 12px;
}

Sticky

Блок, який прилипає

sticky

Прокрути цю сторінку нижче.

Цей маленький блок має position: sticky і top: 12px.

Поки він не дійшов до верхньої межі, він поводиться як звичайний елемент.

Після цього він прилипає.

Fixed

Кнопка в куті екрана

У правому нижньому куті сторінки є маленька кнопка fixed. Вона не рухається під час прокрутки, бо має position: fixed. 

.help-button {
  position: fixed;
  right: 20px;
  bottom: 20px;
}

Тренування

Що можна спробувати самому

  • Пересунути бейдж ближче до лівого краю.
  • Змінити top у sticky-блоку.
  • Переставити fixed-кнопку в лівий нижній кут.
  • Додати z-index і подивитися, як елементи перекриваються.

Прокрутка

Місце для перевірки sticky і fixed
