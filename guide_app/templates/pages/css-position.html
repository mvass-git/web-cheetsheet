Демонстрація

Побачити це в дії

Окрема сторінка показує, як працюють absolute, sticky і fixed на живому прикладі.

Відкрити демо position

position: static

Звичайна поведінка за замовчуванням. Елемент стоїть у нормальному потоці сторінки.

.box {
  position: static;
}

position: relative

Елемент залишається на своєму місці, але його можна трохи зсунути.

.box {
  position: relative;
  top: 10px;
  left: 20px;
}

Часто використовується ще й як “точка прив'язки” для елемента з absolute усередині.

position: absolute

Елемент прибирається зі звичайного потоку і позиціонується відносно найближчого батька з position: relative.

.card {
  position: relative;
}

.badge {
  position: absolute;
  top: 10px;
  right: 10px;
}

position: fixed

Елемент прикріплюється до вікна браузера і не рухається під час прокрутки.

.button-up {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

position: sticky

Елемент поводиться як звичайний, поки не дійде до заданої межі, після чого “прилипає”.

.menu {
  position: sticky;
  top: 0;
}

top, right, bottom, left

Ці властивості задають зміщення елемента для relative, absolute, fixed, sticky.

Властивість Що робить
top Зміщує вниз від верхньої межі.
right Зміщує від правої межі.
bottom Зміщує від нижньої межі.
left Зміщує від лівої межі.

z-index

z-index визначає, який елемент буде поверх іншого.

.menu {
  position: sticky;
  top: 0;
  z-index: 10;
}

Короткий приклад

.card {
  position: relative;
}

.badge {
  position: absolute;
  top: 12px;
  right: 12px;
}

.menu {
  position: sticky;
  top: 0;
  z-index: 10;
}

.help-button {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
