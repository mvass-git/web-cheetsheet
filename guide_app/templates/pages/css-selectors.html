Що таке CSS

CSS відповідає за зовнішній вигляд сторінки: кольори, шрифти, відступи, рамки, фони, розміри та розташування елементів.

  • HTML створює елементи сторінки.
  • CSS змінює те, як ці елементи виглядають.
  • Один CSS-файл можна підключити одразу до багатьох HTML-сторінок.

Будова CSS-правила

h1 {
  color: blue;
  font-size: 40px;
}
Частина Що це Приклад
Селектор Вибирає елемент або групу елементів. h1
Властивість Що саме змінюємо. color
Значення На що змінюємо. blue

Селектори

Селектор Що вибирає Приклад
Тег Усі елементи одного типу. p { color: black; }
.class Усі елементи з певним класом. .card { background: white; }
#id Один унікальний елемент. #title { font-size: 40px; }
* Усі елементи на сторінці. * { box-sizing: border-box; }
a:hover Елемент у певному стані. a:hover { color: red; }
.menu a Теги a всередині блоку .menu. .menu a { text-decoration: none; } 
body {
  margin: 0;
}

.card {
  padding: 20px;
}

#main-title {
  color: darkblue;
}

.menu a:hover {
  color: orange;
}

Кольори і фони

Властивість Для чого Приклад
color Колір тексту. color: darkblue;
background-color Колір фону. background-color: lightyellow;
background Короткий запис для фону. background: #fff8ef;
opacity Прозорість елемента. opacity: 0.8; 
body {
  background-color: #fff8ef;
  color: #1f2937;
}

.banner {
  background: orange;
}

Текст і шрифти

Властивість Для чого Приклад
font-family Шрифт. font-family: Arial, sans-serif;
font-size Розмір тексту. font-size: 20px;
font-weight Товщина тексту. font-weight: bold;
text-align Вирівнювання тексту. text-align: center;
line-height Висота рядка. line-height: 1.6;
text-decoration Підкреслення або його прибирання. text-decoration: none;

Відступи, рамки і модель коробки

Властивість Для чого Приклад
margin Зовнішній відступ. margin: 20px;
padding Внутрішній відступ. padding: 16px;
border Рамка. border: 2px solid black;
border-radius Заокруглення кутів. border-radius: 12px;
box-shadow Тінь. box-shadow: 0 6px 18px rgba(0, 0, 0, 0.15); 
.card {
  margin: 20px;
  padding: 20px;
  border: 1px solid #d0d7de;
  border-radius: 16px;
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08);
}

Розміри

Властивість Для чого Приклад
width Ширина. width: 300px;
height Висота. height: 200px;
max-width Максимальна ширина. max-width: 100%;
min-width Мінімальна ширина. min-width: 220px;

Для адаптивних сторінок часто зручно використовувати %, px, rem.

Display і розташування блоків

Властивість Для чого Приклад
display: block; Елемент займає весь рядок. div { display: block; }
display: inline; Елемент стоїть в одному рядку з текстом. span { display: inline; }
display: inline-block; Елемент у рядку, але з шириною і висотою. a { display: inline-block; }
display: flex; Зручно вирівнювати елементи в ряд або стовпчик. .menu { display: flex; }
gap Відстань між елементами у flex/grid. gap: 16px;
justify-content Вирівнювання по горизонталі. justify-content: center;
align-items Вирівнювання по вертикалі. align-items: center; 
.menu {
  display: flex;
  gap: 12px;
  justify-content: center;
  align-items: center;
}

Корисні властивості, які часто знадобляться

Властивість Для чого Приклад
cursor Вигляд курсора. cursor: pointer;
overflow Що робити, якщо вміст не поміщається. overflow: auto;
list-style Стиль маркерів списку. list-style: none;
transition Плавна зміна стилів. transition: 0.2s;
:hover Стиль при наведенні мишкою. .button:hover { background: darkblue; }

Часті помилки в CSS

  • Забули крапку перед класом: треба .card, а не card.
  • Забули двокрапку після властивості: color: red;.
  • Забули крапку з комою між рядками.
  • Пишуть CSS у файлі, але не підключили його через link в HTML.
  • Задали дуже велику ширину або відступи, через що блок “ламає” сторінку.
  • Плутають margin і padding.
