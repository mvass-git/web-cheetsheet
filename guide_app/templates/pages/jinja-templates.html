Що таке Jinja

Jinja це шаблонізатор для Flask. Він дозволяє вставляти Python-дані в HTML.

  • Показати текст із Python на сторінці.
  • Вивести список через цикл.
  • Показати або сховати блок через умову.
  • Не дублювати однакові частини сторінок.

render_template()

Через render_template() Flask відкриває HTML-шаблон з папки templates і передає в нього дані.

project/
├── app.py
└── templates/
    ├── index.html
    ├── result.html
    └── base.html
from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("index.html", title="Мій сайт")

Змінні в шаблоні

Щоб вставити значення в HTML, використовують {{ ... }}.

<h1>{{ title }}</h1>
<p>Привіт, {{ username }}!</p>
return render_template(
    "index.html",
    title="Головна",
    username="Оля"
)

Умови

Для умов використовують {% if ... %}.

{% raw %}{% if username %}
  <p>Привіт, {{ username }}!</p>
{% else %}
  <p>Ім'я ще не введене.</p>
{% endif %}{% endraw %}

Цикли

Для списків використовують {% for ... %}.

{% raw %}<ul>
  {% for topic in topics %}
    <li>{{ topic }}</li>
  {% endfor %}
</ul>{% endraw %}
return render_template(
    "topics.html",
    topics=["HTML", "CSS", "Flask"]
)

Форма -> інша сторінка

Типовий кейс: взяти дані з форми і показати їх на іншій сторінці.

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def home():
    if request.method == "POST":
        username = request.form.get("name")
        return render_template("result.html", username=username)

    return render_template("form.html")
{% raw %}<!-- result.html -->
<h1>Результат</h1>
<p>Привіт, {{ username }}!</p>{% endraw %}

Саме тут шаблони особливо зручні: сторінка одна, а текст на ній змінюється залежно від того, що ввів користувач.

extends і block

Це спосіб не дублювати каркас сторінки на кожній HTML-сторінці.

{% raw %}<!-- base.html -->
<h1>Мій сайт</h1>
{% block content %}{% endblock %}{% endraw %}
{% raw %}<!-- about.html -->
{% extends "base.html" %}

{% block content %}
  <p>Сторінка про нас</p>
{% endblock %}{% endraw %}

include і часткові шаблони

Якщо шматок коду повторюється, його зручно винести в окремий файл.

{% raw %}<!-- partials/_menu.html -->
<nav>
  <a href="/">Головна</a>
  <a href="/about">Про нас</a>
</nav>{% endraw %}
{% raw %}<!-- base.html -->
{% include "partials/_menu.html" %}{% endraw %}

Так часто виносять меню, підвал сайту, картку товару, повідомлення або інші повторювані блоки.

Що для чого

Конструкція Коли використовувати
{{ value }} Показати змінну на сторінці.
{% if ... %} Показати блок тільки за певної умови.
{% for ... %} Вивести список елементів.
{% extends ... %} Використати спільний каркас сторінки.
{% include ... %} Підключити повторюваний шматок шаблону.
{% endblock %}
