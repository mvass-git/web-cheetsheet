Як це працює

  • Користувач заповнює форму на сторінці.
  • Браузер надсилає дані на маршрут Flask.
  • Flask бере ці дані через request.
  • Сервер може щось із ними зробити і повернути відповідь.

Форма в HTML

<form method="post">
  <label for="name">Ім'я</label>
  <input type="text" id="name" name="name">

  <button type="submit">Надіслати</button>
</form>
  • method="post" означає, що форма надсилає дані на сервер.
  • name="name" це ключ, за яким Flask зможе знайти значення.

Обробка у Flask

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def home():
    if request.method == "POST":
        user_name = request.form["name"]
        return f"Привіт, {user_name}!"

    return render_template("index.html")

Якщо сторінку просто відкрили, повертається HTML. Якщо форму надіслали, Flask бере дані і обробляє їх.

request.form

request.form містить дані, які прийшли з форми.

Запис Що робить
request.form["name"] Взяти значення поля з ім'ям name.
request.form.get("name") Теж взяти значення, але м'якше, без помилки, якщо поля немає.

GET і POST

Метод Коли використовується
GET Коли просто відкриваємо сторінку.
POST Коли надсилаємо дані з форми. 
@app.route("/", methods=["GET", "POST"])

Повний приклад

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def home():
    if request.method == "POST":
        user_name = request.form.get("name")
        return f"Привіт, {user_name}!"

    return render_template("index.html")

if __name__ == "__main__":
    app.run(debug=True)
