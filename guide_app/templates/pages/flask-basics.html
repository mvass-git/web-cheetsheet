Зміст

Структура проєкту Мінімальний app.py Маршрути Що може повертати маршрут Як повернути сторінку url_for() Повний приклад

Структура проєкту

my_project/
  app.py
  templates/
    index.html
    about.html
  static/
    styles.css
    images/
      robot.png
  • app.py містить Flask-код.
  • templates містить HTML-сторінки.
  • static містить CSS, картинки та інші ресурси.

Мінімальний app.py

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("index.html")

if __name__ == "__main__":
    app.run(debug=True)

Запуск: python app.py

Маршрути

Маршрут це адреса сторінки. Наприклад: /, /about, /contacts.

@app.route("/")
def home():
    return "Головна"

@app.route("/about")
def about():
    return "Про нас"

Що може повертати маршрут

Що повертаємо Приклад Що побачить користувач
Звичайний текст return "Привіт!" Текст у браузері.
HTML-рядок return "<h1>Привіт!</h1>" HTML у браузері.
Шаблон return render_template("index.html") HTML-файл із папки templates.

Як повернути сторінку

HTML-сторінки кладемо в папку templates і повертаємо через render_template().

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("index.html")

@app.route("/about")
def about():
    return render_template("about.html")

CSS і картинки треба класти в папку static.

url_for()

url_for() створює правильні посилання на сторінки і ресурси.

Де використовується Приклад Що робить
Посилання на маршрут {{ url_for('about') }} Посилання на сторінку /about.
Посилання на CSS {{ url_for('static', filename='styles.css') }} Посилання на файл зі стилями.
Посилання на картинку {{ url_for('static', filename='images/robot.png') }} Посилання на картинку. 
<link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">

<a href="{{ url_for('about') }}">Про нас</a>

<img src="{{ url_for('static', filename='images/robot.png') }}" alt="Робот">

Повний базовий приклад

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("index.html")

@app.route("/about")
def about():
    return render_template("about.html")

if __name__ == "__main__":
    app.run(debug=True)
<!-- templates/index.html -->
<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <title>Головна</title>
  <link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">
</head>
<body>
  <h1>Ласкаво просимо!</h1>
  <p>Це головна сторінка.</p>
  <a href="{{ url_for('about') }}">Про нас</a>
  <img src="{{ url_for('static', filename='images/robot.png') }}" alt="Робот">
</body>
</html>
