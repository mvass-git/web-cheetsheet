Зміст

px % em rem vw і vh Коли що використовувати Приклад

px

px це пікселі. Фіксований розмір.

h1 {
  font-size: 32px;
}

.card {
  padding: 20px;
}

Добре підходить для рамок, відступів, невеликих точних розмірів.

%

% це відсоток від розміру батьківського елемента.

.box {
  width: 50%;
}

Часто використовується для ширини блоків.

em

em залежить від розміру шрифту поточного елемента.

.card {
  font-size: 20px;
  padding: 1em;
}

Якщо font-size дорівнює 20px, то 1em теж буде 20px.

rem

rem залежить від розміру шрифту кореневого елемента, тобто зазвичай від html.

h2 {
  font-size: 2rem;
}

Зручна одиниця для розмірів тексту і відступів, бо вона більш передбачувана, ніж em.

vw і vh

Одиниця Що означає Приклад
vw Відсоток від ширини вікна браузера. width: 50vw;
vh Відсоток від висоти вікна браузера. height: 100vh;

Часто використовуються для великих блоків на весь екран.

Коли що використовувати

Одиниця Коли зручно
px Точні відступи, рамки, невеликі елементи.
% Ширина блоків відносно контейнера.
em Розміри, які мають залежати від шрифту елемента.
rem Текст і відступи на всій сторінці.
vw, vh Великі секції на екран.

Короткий приклад

body {
  font-size: 16px;
}

h1 {
  font-size: 2rem;
}

.container {
  width: 80%;
  max-width: 900px;
  padding: 20px;
}

.hero {
  min-height: 100vh;
}
