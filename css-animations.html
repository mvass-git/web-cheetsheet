CSS: анімації

Короткий довідник по transition, transform, @keyframes і основних властивостях анімації.

Зміст

transition transform @keyframes animation Параметри animation Що можна анімувати Час і повторення Коли що використовувати Демо

transition

Плавно змінює стиль елемента, коли його стан змінюється, наприклад при наведенні.

.button {
  transition: 0.3s;
}

.button:hover {
  background: darkblue;
}

transform

Дозволяє рухати, обертати, збільшувати або нахиляти елемент.

.card:hover {
  transform: translateY(-8px);
}

.logo {
  transform: rotate(10deg);
}

@keyframes

Описує кроки анімації від початку до кінця.

@keyframes pulse {
  0% {
    transform: scale(1);
  }

  50% {
    transform: scale(1.1);
  }

  100% {
    transform: scale(1);
  }
}

animation

Запускає анімацію, яку описано в @keyframes.

.ball {
  animation: pulse 2s infinite;
}
Частина Що означає
pulse Назва анімації.
2s Тривалість.
infinite Повторювати без кінця.

Параметри animation

Параметр Для чого Приклад
animation-name Назва анімації з @keyframes. animation-name: pulse;
animation-duration Скільки триває одна анімація. animation-duration: 2s;
animation-delay Затримка перед стартом. animation-delay: 1s;
animation-iteration-count Скільки разів повторювати. animation-iteration-count: infinite;
animation-direction Напрямок анімації. animation-direction: alternate;
animation-timing-function Швидкість руху під час анімації. animation-timing-function: ease;
animation-fill-mode Який стан лишити до або після анімації. animation-fill-mode: forwards; 
.box {
  animation-name: pulse;
  animation-duration: 2s;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  animation-timing-function: ease-in-out;
}

Що можна анімувати

Властивість Що змінюється Приклад
transform: translate(...) Рух елемента transform: translateY(-10px);
transform: scale(...) Збільшення або зменшення transform: scale(1.1);
transform: rotate(...) Обертання transform: rotate(15deg);
opacity Прозорість opacity: 0.5;
background-color Колір фону background-color: orange;
color Колір тексту color: darkblue;
width, height Розмір елемента width: 220px;
border-radius Форма кутів border-radius: 50%;
box-shadow Тінь box-shadow: 0 10px 24px rgba(0, 0, 0, 0.2);
letter-spacing Відстань між літерами letter-spacing: 0.1em;

Найчастіше для плавних і красивих ефектів використовують transform та opacity.

Час і повторення

Властивість Приклад
Тривалість animation-duration: 2s;
Затримка animation-delay: 1s;
Кількість повторів animation-iteration-count: infinite;
Швидкість animation-timing-function: ease;

Коли що використовувати

  • transition для простих ефектів при наведенні або натисканні.
  • transform для руху, повороту, масштабу.
  • @keyframes + animation для складніших повторюваних анімацій.
