CSS: розміщення блоків

Короткий довідник по layout: як поставити блоки один під одним, в ряд, у сітку і як їх відцентрувати.

Зміст

Звичайний потік margin: auto inline-block Flex Grid Центрування Швидкі шаблони

Звичайний потік сторінки

За замовчуванням блоки йдуть один під одним зверху вниз.

<div>Блок 1</div>
<div>Блок 2</div>
<div>Блок 3</div>

Якщо нічого не змінювати, саме так і буде виглядати сторінка.

margin: auto

Так часто центрують блок по горизонталі.

.card {
  width: 300px;
  margin: 0 auto;
}

Працює для блоку, у якого є задана ширина.

inline-block

Дозволяє поставити кілька елементів у ряд, але зараз частіше використовують flex.

.box {
  display: inline-block;
  width: 150px;
}

Flex

flex це найзручніший спосіб ставити блоки в ряд або в стовпчик.

.menu {
  display: flex;
  gap: 16px;
}
Властивість Для чого
display: flex Вмикає flex-режим.
flex-direction Напрямок: ряд або стовпчик.
justify-content Вирівнювання по головній осі.
align-items Вирівнювання по другій осі.
gap Відстань між елементами.

Grid

grid зручно використовувати для карток і сіток.

.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Це означає: 3 колонки однакової ширини з відступами між ними.

Центрування

Що треба Рішення
Відцентрувати блок по горизонталі margin: 0 auto;
Відцентрувати текст text-align: center;
Відцентрувати елемент по горизонталі і вертикалі display: flex; justify-content: center; align-items: center; 
.center-box {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 300px;
}

Швидкі шаблони

/* Блок по центру */
.container {
  width: 80%;
  max-width: 900px;
  margin: 0 auto;
}

/* Елементи в ряд */
.row {
  display: flex;
  gap: 16px;
}

/* Картки сіткою */
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

/* Центрування всередині блоку */
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}
