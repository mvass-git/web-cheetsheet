Що таке Flask

Flask це Python-фреймворк для створення веб-застосунків. Простими словами: це програма, яка запускає веб-сервер і показує сторінки в браузері.

  • HTML створює сторінки.
  • CSS оформлює сторінки.
  • Flask віддає ці сторінки браузеру.

Створення застосунку

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Привіт, Flask!"

if __name__ == "__main__":
    app.run(debug=True)
Частина Для чого
from flask import Flask Імпорт класу Flask.
app = Flask(__name__) Створення застосунку.
@app.route("/") Створення маршруту для головної сторінки.
return ... Те, що Flask відправить у браузер.
app.run(debug=True) Запуск сервера.

Як запустити Flask-застосунок

Якщо код збережено у файлі app.py, його можна запустити так:

python app.py

Після запуску у терміналі зазвичай з'явиться адреса на кшталт http://127.0.0.1:5000. Її треба відкрити в браузері.

Маршрути

Маршрут це адреса сторінки на сайті. Наприклад:

  • / це головна сторінка
  • /about це сторінка "Про нас"
  • /contacts це сторінка "Контакти"
@app.route("/")
def home():
    return "Головна сторінка"

@app.route("/about")
def about():
    return "Сторінка про нас"

@app.route("/contacts")
def contacts():
    return "Наші контакти"

Що може повертати маршрут

Що повертаємо Приклад Що побачить користувач
Звичайний текст return "Привіт!" У браузері з'явиться текст.
HTML-рядок return "<h1>Привіт!</h1>" У браузері з'явиться HTML-сторінка.
Шаблон return render_template("index.html") Flask відкриє HTML-файл із папки шаблонів.

Як зробити кілька сторінок

@app.route("/")
def home():
    return "<h1>Головна</h1><a href='/about'>Про нас</a>"

@app.route("/about")
def about():
    return "<h1>Про нас</h1><a href='/'>На головну</a>"

Так можна зробити переходи між сторінками навіть без складної структури проєкту.

Повернення HTML з маршруту

Flask може повернути HTML прямо зі звичайного Python-рядка:

@app.route("/")
def home():
    return """
    <h1>Мій сайт</h1>
    <p>Я вивчаю Flask.</p>
    <a href="/about">Про мене</a>
    """

Це зручно для дуже маленьких прикладів. Для справжніх сторінок краще використовувати окремі HTML-файли.

Шаблони: окремі HTML-файли

У Flask часто створюють папку templates і кладуть туди HTML-файли. 

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("index.html")

@app.route("/about")
def about():
    return render_template("about.html")

Тоді HTML лежить окремо, і код Python стає чистішим.

Часті помилки

  • Забули імпортувати Flask або render_template.
  • Написали маршрут, але не поставили @app.route(...).
  • Дві функції мають однакову назву.
  • У return повертається неправильний текст або зламаний HTML.
  • Запустили не той файл або не зберегли зміни перед запуском.
  • HTML-файл для render_template лежить не в папці templates.

Повний базовий приклад

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return """
    <h1>Ласкаво просимо!</h1>
    <p>Це головна сторінка.</p>
    <a href="/about">Про нас</a>
    """

@app.route("/about")
def about():
    return """
    <h1>Про нас</h1>
    <p>Ми вивчаємо Python Flask.</p>
    <a href="/">На головну</a>
    """

if __name__ == "__main__":
    app.run(debug=True)
